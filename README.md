# Hue-CLI

Command line app for controlling Philips Hue Lights.

```
hue 1.0.0
Copyright (C) 2018 hue

  -b, --bridges     List bridges on the network
  -r, --register    Register with a Hue Bridge. Registration usually happens automatically, you should only need to use to fix a
                    broken registration.
  -i, --ip          IP Address of the Hue Bridge. Will default to the first bridge found.
  -w, --wait        Wait before exiting.
  --lights          List lights
  --help            Display this help screen.
  --version         Display version information.
```
